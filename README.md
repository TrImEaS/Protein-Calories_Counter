# 🥗 Protein Calories Counter

Una aplicación potente y sencilla para seguir tus objetivos nutricionales, enfocada principalmente en proteínas, calorías y azúcar.

## ✨ Funcionalidades Principales

- **Tablero de Control Dinámico:** Visualiza rápidamente tu progreso diario con gráficos intuitivos.
- **Gestión de Alimentos Personalizados:** Crea, edita y administra tu propia base de datos de alimentos con detalles de peso, azúcar, proteínas y calorías.
- **Escaneo de Códigos de Barras:** Registra alimentos de manera rápida usando la cámara de tu dispositivo.
- **Múltiples Porciones:** Calcula automáticamente los nutrientes según la cantidad de porciones consumidas.
- **Historial Detallado:** Revisa tus registros pasados para mantener un seguimiento de tu evolución.
- **Personalización Total:** Define tus metas diarias y elige entre temas claro u oscuro según tu preferencia.
- **Soporte Multi-idioma:** Disponible en Español e Inglés.

## 🛠️ Tecnologías Utilizadas

Desarrollada con las últimas tecnologías en el ecosistema móvil:

- [Expo](https://expo.dev/) & [React Native](https://reactnative.dev/)
- [React Native Paper](https://reactnativepaper.com/) (Componentes de UI)
- [Zustand](https://github.com/pmndrs/zustand) (Gestión de estado)
- [Tanstack Query](https://tanstack.com/query)
- [Tailwind CSS (via twrnc)](https://github.com/origami-studio/twrnc)
- [Expo Camera](https://docs.expo.dev/versions/latest/sdk/camera/) (Para el escaneo de códigos)

## 📦 Cómo Empezar

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/trimeas/Protein-Calories_Counter.git
    ```
2.  **Instala las dependencias:**
    ```bash
    npm install
    ```
3.  **Inicia la aplicación:**
    ```bash
    npx expo start
    ```
